# Snap-Caption

A professional-grade, zero-setup video extraction pipeline with AI transcription powered by Groq.

## Usage

1. **Deploy to Hugging Face Spaces** as a Docker space.
2. **Set Secrets**: Go to your Space Settings and add `GROQ_API_KEY` (your API key from Groq).
3. **Enjoy!**
